SQL for Petroleum Engineers

A hands-on SQL portfolio focused on solving production and drilling engineering problems using realistic oil and gas datasets. This repository is part of the Digital Petroleum Engineering Academy.

## About

This repo documents my Phase 2 journey through the Digital Petroleum Engineering Academy (DPE) — moving from Excel-based production analytics (Phase 1) into database querying with SQL.

Each week's folder contains:
- SQL query scripts solving real production/drilling engineering problems
- A technical report interpreting the results from an engineering perspective
- Supporting documentation (schema diagrams, README notes)

## Skills Demonstrated

- Relational database fundamentals
- `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `DISTINCT`
- Joins across well, production, and drilling tables
- Aggregate functions (`AVG`, `SUM`, `GROUP BY`)
- Translating engineering problems into SQL logic
- Interpreting query results from a production/drilling engineering perspective
